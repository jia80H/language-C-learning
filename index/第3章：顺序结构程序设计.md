# 第3章: 顺序结构程序设计

## 3.1 c语句概述
c语句分为以下五类:
- 表达式语句, 如:x=9;
- 函数调用语句, 如:函数名(参数);
- 控制语句
    - 选择结构控制语句: if()...else...  switch()...case...
    - 循环结构控制语句: while()... do...while()  for()...
    - 转向语句: break continue goto return
- 复合语句,如:{int x,y;  x=y+x; y-=1;}
- 空语句: 仅含有一个';'
